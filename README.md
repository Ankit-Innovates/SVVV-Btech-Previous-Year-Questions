# 📚 SVVV Question Papers Repository

This repository contains semester-wise and subject-wise **MST (Mid Semester Test)** and **End Semester** question papers.

> 📚 A centralized collection of SVVV B.Tech previous year question papers
> Organized semester-wise and subject-wise for easy access.

---

## 🎯 Coverage

* Semester 2 to Semester 8
* Subjects: Data Structures, DBMS, Machine Learning, etc.
* Exam Types:

  * MST (Mid Sem)
  * End Semester
